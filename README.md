# Thesis_OMG
# Repository Contents

This repository contains two Jupyter notebooks:

1. **Model notebook** (`Thesis_model.ipynb`): contains the BTM topic modelling pipeline and the decision tree model, including feature construction, training, and evaluation.
2. **Data analysis notebook** (`Thesis_OMG analysis (group phase).ipynb`): contains the exploratory data analysis, which is a group-based contribution shared across the team's individual thesis projects.
